Brownstein Labrid Tree — Handoff for FishView Image Agent

Purpose: The FishView image server agent needs to populate family image directories for the 4 labrid project groups. Each group needs lateral-view photographs matched to species in the Brownstein et al. (2025) phylogeny. This handoff provides the tree, the species lists, and the matching rules.

The tree

Citation: Brownstein, C. D., et al. (2025). Phylogenomics establishes an Early Miocene reconstruction of reef vertebrate diversity. Science Advances, 11, eadu6149. https://doi.org/10.1126/sciadv.adu6149

Dryad: https://doi.org/10.5061/dryad.f7m0cfz5x

File: labs/demo-labrid-pcm-week8/data/labrid-time-calibrated-tree.tre (NEXUS format)

Stats: 415 tips, time-calibrated (GTR + node-dating, 13 fossil calibrations), root age 70.2 Ma. Covers all major labrid subfamilies + outgroups.

The 4 labrid project groups

These 4 of the 7 class project groups use subclades of the Brownstein tree. The other 3 groups (Balistoidea, Pomacanthidae, Acanthuridae) use family-specific trees from other sources.

Tip label format

Tree tip labels have the format: Genus_speciesVOUCHER

Examples:

Scarus_ghobbanF204
Cirrhilabrus_scottorumF628KB9
Halichoeres_melanotisF72
Bodianus_rufus_F112

To match images to tree tips: 1. Extract genus + species: sub("([A-Z][a-z]+_[a-z]+).*", "\\1", tip_label) 2. This gives Scarus_ghobban, Cirrhilabrus_scottorum, etc. 3. Match to image filename: genus-species.jpg (lowercase, hyphenated) on FishView

Beware: - Some species have multiple tips (different specimens/vouchers). The image agent should provide ONE image per species — the R script will handle deduplication. - Some tip labels have underscores in unexpected places (e.g., Bodianus_rufus_F112 — the species is rufus, not rufus_F112). The regex above handles this.

What the image agent needs to produce per family

For each of the 4 labrid family directories on FishView:

~/fishview-data/families/<family>/
├── images_fishwise/     ← FishWise lateral views (preferred)
├── images_fishbase/     ← FishBase photos
├── images_inaturalist/  ← iNaturalist CC photos
└── segmented/           ← alpha-masked PNGs (post-processing)

Minimum viable set for Demo 7 Part 2: 10 species per group, spanning genera. The tip label CSVs list every species in the tree — the agent should prioritize species that are: 1. In the tree (required) 2. Visually distinctive (good for color analysis) 3. Spread across genera (not all one genus)
