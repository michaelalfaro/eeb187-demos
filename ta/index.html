Demo 7 — Mapping Color Patterns onto a Phylogeny

EEB 187 · Week 8 · Wednesday May 20, 2026

TipBackground reading

Frédérich, B., Mittelheiser, L., Gillet, A., Hodge, J. R., Laudet, V., & Dornburg, A. (2026). Rapid and repeated evolution of pigmentation patterns in reef fishes. BMC Biology, 24(46). DOI · PDF (open-access, CC BY-NC-ND)

This is the paper Demo 7 is built around. Skim it the night before — the worksheet and the FishView review session are the hands-on companions to its question: do color patterns track phylogeny, or do they evolve so fast that relatedness predicts almost nothing?

Overview

In this demo you’ll learn to map color pattern traits onto a phylogenetic tree and test whether closely related fish species have more similar color patterns than expected by chance. The demo has two halves:

  • Part 1 is R-driven: you’ll load Frédérich et al.’s coded color-pattern matrix for the Acanthuridae (surgeonfishes), map it onto the family’s time-calibrated phylogeny, and run two formal tests of phylogenetic signal.
  • Part 2 is server-driven: each group will log into FishView, pick exemplar images for the species in their assigned family, and download a bundle that pairs those images with the tree. An R helper turns the bundle into a tree-with-photos figure.

Methods: contMap, Blomberg’s K, Pagel’s λ, discrete trait mapping, ape::keep.tip() on a per-group subtree.

How to download all four files at once

Option 1 (easiest): Right-click each download link above → “Save Link As…”

Option 2 (R one-liner): paste this block into the RStudio Console after setwd()-ing into your demo working folder:

dir.create("data/frederich", recursive = TRUE, showWarnings = FALSE)
dir.create("scripts",        recursive = TRUE, showWarnings = FALSE)

# Part 1 data
download.file("https://michaelalfaro.github.io/eeb187-demos/data/frederich/acanthuridae-tree.tree",
              "data/frederich/acanthuridae-tree.tree")
download.file("https://michaelalfaro.github.io/eeb187-demos/data/frederich/acanthuridae-color-patterns.csv",
              "data/frederich/acanthuridae-color-patterns.csv")

# Part 1 walkthrough script (optional — only if you prefer pasting chunks vs. opening the worksheet)
download.file("https://michaelalfaro.github.io/eeb187-demos/scripts/walkthrough-part1.R",
              "walkthrough-part1.R")

# Part 2 helper (needed for the tree-with-exemplar-images figure)
download.file("https://michaelalfaro.github.io/eeb187-demos/scripts/part2-tree-with-images.R",
              "scripts/part2-tree-with-images.R")
Note

The Part 2 helper (scripts/part2-tree-with-images.R) is sourced by the worksheet’s Part 2 chunk via source("scripts/part2-tree-with-images.R"). The worksheet’s own Part 2 chunk also re-downloads this file as its first step, so you don’t have to do it manually — but if you’d rather have everything ready before class starts, the one-liner above pulls it for you.
